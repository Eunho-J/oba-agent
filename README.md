# OBA Agent

Public pitch page for OBA Agent, a self-evolving personalized agent focused on agent-experience-conditioned memory recall.

- The core problem: agents can store memory flexibly, but recall memory too uniformly.
- The core evolution target: not only skills or workflows, but the recall policy that decides how memory is associated.
- whisper.cpp/API STT adapter for speech input.
- EXAONE as the post-STT Korean text cleanup and expression layer.
- A lightweight main agent API as the reasoning and tool-execution layer.
- MISO workflows as the ontology and memory layer for expression and reasoning.
- ApiFuse and ggui as action and UI surfaces.

GitHub Pages serves the static site from the repository root.
